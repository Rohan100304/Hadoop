# Linux Commands, Pig and Hive Queries

This repository contains basic notes and examples related to **Linux commands**, **Apache Pig**, and **Apache Hive**.

## Contents

- **Basic Linux Commands**
  - A list of commonly used Linux commands
  - Each command includes a short description explaining its purpose

- **Pig Queries**
  - Sample Pig Latin queries using a sample dataset
  - Includes explanations of operations such as loading data, filtering, grouping, and aggregation

- **Hive Queries**
  - Example Hive SQL queries on a sample dataset
  - Includes table creation, data loading, filtering, and grouping queries with explanations

This repository is intended as a **simple reference for beginners learning Linux and Big Data tools like Pig and Hive**.
